Voici le Bot de DevArea ! Il apporte de nouvelles fonctionnalités au serveur.

Vous pouvez utiliser toutes les commandes ci-dessous:
//help -> renvoie la liste des commandes disponibles.
//ping -> renvoie la latence du bot en milli-secondes.
//meetup -> permet de mettre en place des conférences/regroupements vocaux autour d'un sujet.
//start -> renvoie les prérequis nécessaires pour débuter dans les langages disponibles.
//rank -> affiche votre rang et votre xp sur le serveur.
//leaderboard -> affiche la liste des 5 membres ayant le plus d'xp du serveur.
//devhelp -> permet de mentionner les développeurs possédants le rôle Dev'Helper dans le channel, utile si personne n'a répondu à votre demande
//ask -> affiche les informations nécessaires pour poser une question efficacement.
//mission -> permet de gérer vos propositions de missions.

creationMission -> permet de créer une mission dans le channel missions-payantes, /!\: uniquement dans ce channel.

Toutes les commandes commencants par "//" doivent être utilisées dans les salons du serveur !
